Para execução das tarefas abaixo, use o projeto AvaliacaoII_Estrela

1) Criar uma servlet para receber a requisição do formulário presente em index.html
        A servlet deve ter o nome adequado para que o formulário funcione sem nenhuma alteração.
Escreva o código adequado para inserir as informações no banco de dados.

Deve ser exibido ao usuario o resultado da operação.

2) Em outra página, você deverá implementar uma listagem de todos os registros armazenados no banco.

É indispensavel o uso de JSP na solução.
