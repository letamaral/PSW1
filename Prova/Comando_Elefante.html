    • Objetivo geral:

    Voce devera implementar um sistema que exiba uma pergunta ao usuario e dê a ele a oportunidade de digitar uma resposta. Em seguida o usuario deve ser capaz de enviar essa resposta ao sistema que deverá criticar essa resposta. Caso esteja correta, o sistema deve informar que o usuário acertou. Se a resposta estiver errada, o sistema deve informar a diferença entre a resposta certa e a resposta do usuário, conforme exemplo na imagem.

    Pergunta: Qual o peso de um elefante adulto? (Resposta: 4500)

    Objetivos específicos:

    1. Crie um novo projeto WEB no Netbeans
    2. Crie um formulario HTML no qual o usuario poderá ler a pergunta e escrever a resposta.
    3. Crie uma servlet que receberá a resposta enviada pelo formulario HTML.
    4. Escreva código suficiente na servlet para
      1. recuperar o valor digitado pelo usuario
      2. verificar se a resposta está certa ou errada.
      3. caso esteja certa, exibir mensagem de parabens ao usuario
      4. caso esteja errada:
        1. calcular a diferença entre a resposta certa e a resposta informada
        2. exibir mensagem de erro para o usuario.
        3. exibir a diferenca entre a resposta certa e a resposta informada.
    5. É indispensavel que seu codigo esteja funcionando de acordo com o descrito. Sua avaliacao somente sera considerada entregue apos ser apresentada ao professor.
